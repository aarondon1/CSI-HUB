Dolphin Finder API

Welcome to the Dolphin Finder API. Below are the available endpoints:

Profile Endpoints

GET /profile/<user_id>/ - Get a user profile

POST /create-profile/ - Create a new profile

PATCH /profile/<user_id>/ - Update a profile

GET /me/ - Get current user profile

Project Endpoints

GET /homepage/ - Get all projects (with optional ?query= parameter)

POST /create-project/ - Create a new project

GET/PUT/DELETE /projects/<project_id>/ - Get, update, or delete a project

GET /user-projects/<user_id>/ - Get projects by a specific user

Join Request Endpoints

POST /join-request/ - Create a join request

PATCH /join-request/<request_id>/status/ - Update join request status

GET /join-request/user/<user_id>/ - Get received join requests

GET /join-request/sent/ - Get sent join requests
